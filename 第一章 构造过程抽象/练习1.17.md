
### 1,迭代法：

```Scheme
(define (double n)
    (+ n n))

(define (halve n)
    (/ n 2))

(define (iter c a b)
  (cond ((= b 0) c)
        (odd? b) (iter (+ c a) a (- b 1))
        (else (iter c double(a) halve(b)))))

(define (fast-multi x n)
  (iter 0 x n))
```

### 2,递归法:
利用类似书本 30 页的 fast-expt 的技术，写出使用对数步数求乘积的函数
看着代码很容易看懂，但是就是想不出答案。。。
```Scheme
(define (double n)
    (+ n n))

(define (halve n)
    (/ n 2))
    
(define (multi a b)
    (cond ((= b 0)
            0)
          ((even? b)
            (double (multi a (halve b))))
          ((odd? b)
            (+ a (multi a (- b 1))))))
```
