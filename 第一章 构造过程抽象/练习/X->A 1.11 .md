### 改正代码
```

#lang racket
(define (f n)
  (cond ((= n 0) 0)
        ((= n 1) 1)
        ((= n 2) 2)
        (else (+ (* 3 (f (- n 3)))
                 (* 2 (f (- n 2)))
                 (* 1 (f (- n 1)))))))


```
